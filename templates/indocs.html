Input Format

Our endpoint URL is:

http://seng3011laser.com/api/v1/PageData

Parameters
Name Required Description Type Example
Company Required This is the company name or unique ID that the user would like to search for. The input is not case sensitive and will accept both upper and lower case values. String WOW.AX, Woolworths, AMZN, Amazon
Start Required This is the time selected from which you wish to retrieve data from. (Must be in the format – “yyyy-MM-ddTHH:mm:ss:SSSZ”) String “2015-10-01T08:45:10.295Z”
End Required This is the time selected from which you wish to retrieve data to. (Must be in the format – “yyyy-MM-ddTHH:mm:ss:SSSZ”) String “2015-11-01T19:37:12.193Z”
Fields Optional (Minimum 1 field is required) The fields allows users to only receive statistics in the fields they have listed. The fields accept each variable separated by a comma. The fields include: id, name, website, description ,category, fan_count, post_type, post_message, post_created_time, post_like_count, post_comment_count String id, name, website, description, category, fan_count
Definitions for terms in fields:

- Id: the ID of the company interested assigned by Facebook

- Name: the name of the company interested

- Website: the website of the company interested

- Description: A description on the company interested

- Category: Determines which category the company is under

- Fan_count: The amount of people who like and follow the company

The following definitions are for specific statistics on particular posts the company has published.

- Post_type: The type of post

- Post_message: The content of the post

- Post_created_time: The time the post was created

- Post_like_count: How many likes the post has accumulated

- Post_comment_count: How many comments the post has
