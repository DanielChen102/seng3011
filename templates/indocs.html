Input Format

  • V2
  • V1

Our endpoint URL is:

http://seng3011laser.com/api/v2/PageData

All input parameters should be in lower case and spaces are not accepted (Should be replaced with a hyphen or underscore).

Parameters
Name Required Description Type Example
Company Required This is the Facebook page name String woolworths, mcdonalds
StartDate Required This is the time selected from which you wish to retrieve data from. (Must be in the format – “yyyy-MM-ddTHH:mm:ss:SSSZ”) String 2015-10-01T08:45:10.295Z
EndDate Required This is the time selected from which you wish to retrieve data to. (Must be in the format – “yyyy-MM-ddTHH:mm:ss:SSSZ”) String 2015-11-01T19:37:12.193Z
Fields Optional (Minimum 1 field is required) The fields allows users to only receive statistics in the fields they have listed. The fields accept each variable separated by a comma. The fields include: id, name, website, description ,category, fan_count String id, name, website, description, category, fan_count
Post Fields Optional Fields related to posts on the page. Post fields include: post_type, post_message, post_created_time, post_like_count, post_comment_count String post_type,post_message

An example query could be: 

 
 http://seng3011laser.com/api/v2/PageData?company=woolworths&startdate=2015-10-01T08:45:10.295Z&enddate=2015-11-01T19:37:12.193Z

This query is requesting statistics from the company: 'woolworths' from October 1st 2015 at 8:45 until November 1st 2015 at 19:37

To add fields to the this example, such as requesting the company's website, description and fan_count, we append the fields onto the end like so 

 
 http://seng3011laser.com/api/v2/PageData?company=woolworths&startdate=2015-10-01T08:45:10.295Z&enddate=2015-11-01T19:37:12.193Z&fields=description,fan_count,website

To query for post information, append ,post.fields() with the desired fields (seperated by commas) in the brackets, to the end of the fields

 
 http://seng3011laser.com/api/v2/PageData?company=woolworths&startdate=2015-10-01T08:45:10.295Z&enddate=2015-11-01T19:37:12.193Z&fields=description,post.fields(post_message)
Definitions for terms in fields:

- Id: the ID of the company interested assigned by Facebook

- Name: the name of the company interested

- Website: the website of the company interested

- Description: A description on the company interested

- Category: Determines which category the company is under

- Fan_count: The amount of people who like and follow the company

The following definitions are for specific statistics on particular posts the company has published.

- Post_type: The type of post

- Post_message: The content of the post

- Post_created_time: The time the post was created

- Post_like_count: How many likes the post has accumulated

- Post_comment_count: How many comments the post has
