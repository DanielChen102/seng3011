Output Format

Below is an example of a response from the API that requested for data from Woolworths. The API outputs this as a JSON file. 

{
     "Facebook Statistic Data": [
     {
        "PageId"": <woolworths>,
        "InstrumentIDs": <WOW.AX>,
        “CompanyNames”:<woolworths>,
        "PageName": <woolworths>,
        "Website": <www.woolworths.com.au>,
        "Description ":< Welcome to the official Facebook Page for the fresh food people.We’re here Monday to Friday 9am-9pm AEST and Saturday/Sunday 10am to 5pm AEST.>,
        "Category": <Retail Company>,
        "fan_count": <1,057,425 >,
        "posts [
        {
            "post_id": <>,
            "post_type": <>,
            "post_message": <>,
            "post_created_time": <>,
            "post_like_count": <>,
            "post_comment_count": <>
     }
]
Field Name Description
PageId The id of the page on Facebook. E.g The pageId of Woolworths is Facebook.com/woolworths
InstrumentId The unique code given to the company
CompanyNames Official name of the company
PageName The name that is displayed on the Facebook page
Website The comapany's official website
Description The text inside the company page's description section
Category The category of the page on Facebook
FanCount How many fans like the page
Posts A collection of posts with and each post's information.

Log File

The API also outputs a log file which includes the following data

  • Developer name
  • Module name and version
  • Parameters passed
  • Success or error indication
  • Error messages if an error occured
  • Start time, end time,elapsed time and output file name if successful
